# Apps

UI applications vibe-coded by AI agents. Each app is a subdirectory.

**Types:**
- Single HTML files (quick visualizations)
- React apps (complex tools)
- DSL-based composable apps

**Key apps to build:**
- `viewer/` - Core file navigation + windowing system
- `markdown-canvas/` - Notion-like editor
- `ai-manager/` - Agent monitoring, cost tracking
